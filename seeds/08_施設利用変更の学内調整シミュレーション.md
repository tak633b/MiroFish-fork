# 施設利用変更の学内調整シミュレーション

**用途**: 学内交渉の戦略立案 / 根回し順序の最適化 / 縦割り組織の合意形成パターン検証

---

## 🌱 リアリティシード

**【研究者による実験室の用途変更申請】**

舞台：総合大学の理系学部棟。研究者から「現在の教育用実験室の一角を、自身の研究プロジェクト（生体材料を使った実験）に使いたい」という申請が提出された。課長（蓮村）は担当として、この申請を処理しなければならない。

### 登場人物

| 名前 | 役職 | 特性 |
|------|------|------|
| 蓮村（課長） | 教育支援担当課長 | 安全性と現実的な落とし所を探る。調整役として中立を保ちたい |
| 細田（申請研究者） | 准教授 | 研究の自由と利便性を強く主張。「他大学ではできている」と引き合いに出す |
| 教務部担当 | 教育課程管理 | 教育活動への影響と授業使用との競合を懸念 |
| キャンパス企画部 | 施設管理 | 施設規程・改修コスト・消防法への適合を管理。変更には消極的 |
| 安全管理委員会 | リスク管理 | 生体材料のリスク評価を厳密に行う原則論者。「前例がない」を重視 |
| 事務局長 | 最終承認権限者 | 政治的判断を優先。研究者の声と規程論のバランスを取りたい |

### 経緯

- 細田准教授が半年前から非公式に「使わせてほしい」と繰り返し打診
- 課長が安全管理上の問題を指摘したところ、准教授は「ならば正式申請する」と書面提出
- 安全管理委員会は2007年の類似事例を根拠に難色を示している
- キャンパス企画部は「改修工事が必要になれば費用負担は誰か」と懸念
- 教務部は「授業利用との共存は難しい」と反対
- 事務局長は「研究者の要望も大切」と示唆しており、判断が読めない

## 🔮 模擬プロンプトワード

課長はこの申請をどう処理すべきか。
各部署が縦割りで異なる論理で動く中、どの順番で根回しをすれば合意が取れるか。
「全面不可」以外の妥協点（条件付き許可、別スペース提案など）は生まれるか。
研究者が強引に進めようとした場合、どうなるか。

## 📌 想定エージェント構成

| エージェント | 初期立場 | 行動傾向 |
|------------|---------|---------|
| 蓮村（課長） | 中立・調整 | 各部署の懸念を収集し、条件付き承認の可能性を探る |
| 細田（研究者） | 強く希望 | 「研究の自由」「他大学の実績」を繰り返す。断られると上に訴える |
| 教務部 | 消極的反対 | 授業利用との競合を理由に慎重。根拠を求められると曖昧になる |
| キャンパス企画部 | 反対（費用懸念） | 改修コストと責任負担を最大の懸念として提示する |
| 安全管理委員会 | 反対（原則論） | 「前例がない」「2007年の経緯」を根拠に全面不可を主張しやすい |
| 事務局長 | 曖昧・様子見 | どちらの意見にも「それも一理ある」と言いがち。最終判断を先延ばしにする |

---

## 📊 推奨シミュレーション設定

- ラウンド数: 20〜35
- 注目指標: 合意形成の可否 / 誰に先に話すと通りやすいか（根回し順序） / 研究者が強行した場合のシナリオ分岐
- 発展活用: 学内折衝の戦略立案・断り方の文書化・条件付き承認の枠組み設計に転用可能
