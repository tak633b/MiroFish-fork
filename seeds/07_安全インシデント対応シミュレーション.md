# 安全インシデント対応シミュレーション

**用途**: 緊急時対応プロトコルの検証 / 情報伝達の歪み発見 / ボトルネック特定

---

## 🌱 リアリティシード

**【化学実験中の薬品被曝事故】**

舞台：大学の化学実験室。午後の実験授業中、学生が試薬の取り扱いを誤り、皮膚に薬品が付着した。直ちに処置が必要だが、当事者以外に何人かがその場にいる。課長（蓮村）は別の会議に出席中で、事故発生をすぐには知らない。

### 登場人物

| 名前 | 役職 | 特性 |
|------|------|------|
| 蓮村（課長） | 教育支援担当課長 | 迅速報告と被害最小化を優先。会議から戻り次第対応に入る |
| 坂本（嘱託職員） | 化学実験室担当 | 現場対応に専念するあまり、上への報告が遅れがち |
| 木村（担当教員） | 化学実験授業担当 | 授業の続行と自分の責任回避を優先してしまう面がある |
| 安全管理室担当 | 大学安全管理室 | 規程に従い事故記録・調査を要求。手続きが複雑 |
| 田中（学生・被害者） | 2年生 | 処置より保護者への連絡を気にしている。軽視されると不満が膨らむ |
| 学務課担当 | 学生担当窓口 | 保護者対応・公式記録の担当。正確な情報がないと動けない |

### 経緯

- 14:30 実験中に学生・田中が試薬（弱酸）を手に付着
- 嘱託職員・坂本が水で洗浄対応。軽症と判断
- 担当教員・木村は「授業を止めたくない」と実験を継続
- 安全管理室への報告は誰も着手していない
- 田中は「保護者に連絡してほしい」と坂本に訴えている
- 蓮村は15時の会議から戻ってくる

## 🔮 模擬プロンプトワード

事故発生後30分の組織の動きを推演してほしい。
誰が最初に情報を報告し、誰が握りつぶすか。
責任帰属をめぐって誰が動き、誰が静観するか。
課長が戻ってきた時点での「状況」と「情報の歪み」を明らかにしてほしい。

## 📌 想定エージェント構成

| エージェント | 初期心理状態 | 行動傾向 |
|------------|------------|---------|
| 蓮村（課長） | 情報収集フェーズ | 戻り次第、状況全体を把握しようとする。正確な情報を求める |
| 坂本（嘱託職員） | 対応に追われている | 現場処置は適切。ただし上への報告は後回し |
| 木村（教員） | 自己防衛モード | 「大したことない」と発言し、授業を続けようとする |
| 安全管理室 | 規程遵守 | 正式な報告書がないと動けない。手続き論を繰り返す |
| 田中（学生） | 不安・軽視への怒り | 「誰も自分を気にしてくれない」という認識が強まる |
| 学務課 | 情報待ち | 保護者連絡の準備をしたいが、正確な情報がない |

---

## 📊 推奨シミュレーション設定

- ラウンド数: 20〜30
- 注目指標: 情報がどの経路で歪むか / 誰がボトルネックになるか / 課長介入前後での対応の変化
- 発展活用: 緊急時連絡フローの改訂・初動対応マニュアルの作成に直接転用可能
