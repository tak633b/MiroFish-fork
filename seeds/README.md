# MiroFish Seeds — 大学管理職シナリオ集

バルがたかさん（早稲田大学 教育支援担当課長）の実務に基づいて設計したシミュレーションシード集。

## シナリオ一覧

| ファイル | テーマ | 用途 |
|---------|--------|------|
| [06_嘱託採用選考](./06_嘱託採用選考シミュレーション.md) | 最終候補2名の採用選考 | 評価バイアス・合意形成プロセスの検証 |
| [07_安全インシデント対応](./07_安全インシデント対応シミュレーション.md) | 実験中の薬品被曝事故 | 緊急時の情報伝達と責任帰属の検証 |
| [08_施設利用変更](./08_施設利用変更の学内調整シミュレーション.md) | 研究者による実験室用途変更申請 | 縦割り組織の根回し戦略・合意形成の検証 |

## 共通の特徴
- 登場人物はすべて実在するロール（鹿倉・阿知波・細田など）がモデル
- たかさん役（蓮村課長）がエージェントの一人として参加
- 「合意形成がどう進むか」「誰がボトルネックか」の発見に最適化

## 使い方
1. MiroFishのプロジェクト作成画面でシナリオ本文を貼り付け
2. エージェント設定は「想定エージェント構成」表をそのまま入力
3. ラウンド数は「推奨シミュレーション設定」を参照
