# 嘱託採用選考シミュレーション

**用途**: 採用選考の合意形成プロセス検証 / 評価バイアスの発見 / 交渉戦略の立案

---

## 🌱 リアリティシード

**【化学実験室嘱託職員 最終選考】**

舞台：大学の教育支援部門。化学実験室の嘱託職員（任期付きスタッフ）が退職し、後任選考が最終局面を迎えた。候補者2名が残り、来週に採否を決定しなければならない。

### 登場人物

| 名前 | 役職 | 特性 |
|------|------|------|
| 蓮村（課長） | 教育支援担当課長 | 組織全体の安定・コスト・長期安定性を重視。現場と人事の橋渡し役 |
| 鹿倉（派遣会社） | リクルートスタッフィング担当 | 候補者を推薦した立場。採用成立を急いでいる。候補者Bを強く推す |
| 福田（担当教員） | 化学実験授業担当 | 専門的実技力を最重視。実験安全管理に厳しい。候補者Aを推す |
| 阿知波（人事課） | 人事課担当者 | 規程遵守・前例踏襲。週30時間超勤務の候補者Bに制度上の疑念あり |
| 候補者A | 前職：他大学実験技術員 | 経験豊富・実技力高い。ただし年齢55歳・給与希望が高め |
| 候補者B | 前職：民間製薬会社 | 実験経験あり・若く柔軟・コスト低め。ただし大学業務は未経験 |

### 経緯

- 前任者は2月末退職。4月開講に向けて急いでいる
- 候補者Aは前回面接で専門知識をアピール、教員の評価が高い
- 候補者Bは鹿倉が積極的に推薦。コストと柔軟性が魅力
- 人事課は候補者Bの「週30時間超勤務」希望が規程に抵触する可能性を指摘

## 🔮 模擬プロンプトワード

各ステークホルダーがそれぞれの利害と価値観で動く中、課長は誰を採用すべきか。
派遣会社の営業圧力、教員の専門性重視、人事の規程論が衝突した時、
合意はどのように形成されるか。あるいは形成されないか。

## 📌 想定エージェント構成

| エージェント | 初期立場 | 行動傾向 |
|------------|---------|---------|
| 蓮村（課長） | 中立・調整役 | 各意見を聞きながら落とし所を探る。最終決定権を持つ |
| 鹿倉（派遣） | 候補者B推し | 「柔軟性」「コスト」を繰り返し強調。急かす発言が多い |
| 福田（教員） | 候補者A推し | 実技力・安全管理の実績を根拠に主張。感情的になりやすい |
| 阿知波（人事） | 候補者B懸念 | 規程への適合を優先。どちらの候補にも慎重 |
| 候補者A | 採用されたい | 経験と実績を丁寧にアピール。過去の事例を多く引用 |
| 候補者B | 採用されたい | 積極的な姿勢を見せる。大学業務の未経験を補おうとする |

---

## 📊 推奨シミュレーション設定

- ラウンド数: 15〜25
- 注目指標: 課長への影響力が最も強いエージェントはどれか / 規程論と現場論の拮抗 / 最終合意の形成パターン
- 発展活用: 採用面接の評価項目設計・選考プロセスの改善提案に直接転用可能
